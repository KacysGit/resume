Color Catalog


I wanted to learn how to program in React Native since it is compatible with web development, android devices, and IOS devices. To complete this goal, I followed the LinkedIn learning course Learning React Native by Alex Banks. Throughout this course, I follow Mr. Banks' step by step guide to making a Color Catalog app using React Native. By the end, I had fully programmed an app that lets the user add buttons to a list to change the background color.

Hercules Fitness App


From August 2023 through November 2023, I worked with Kaleb Perez, Shelby Cumings, Julio Cantu, and Aaron Parks to make a Fitness app called Hercules. Using React Native, we were able to create an app that lets the user create trackers (for example to monitor water intake of sleep intake) and also monitor their meals and workouts. Each button in the app was transparent with a graph over it so that the user could see their progress through the graph and click on it to navigate to the next screen.
